#DDYCaptchaView

* 本机生成随机字符串,然后加上干扰线

* TOTP生成特定位数随机码并且服务器共用同样算法生成同样随机码，（Google Authenticator使用此方法）

* 从左边滑动到最右边，根据各种行为(点击点，时间等)判断是不是人类

* 将一个控件拖动到随机出现的圆内(火币虚拟货币交易App采用此方法) (开发中)

* 将镂出来的滑块拖回镂空处（微信使用此方法）(开发中)

* 将字符绘制在干扰图上，根据用户点击选择(开发中)
